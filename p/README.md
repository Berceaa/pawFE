# Pawsentials redesign

Am refăcut interfața într-un stil inspirat de magazine mari de pet retail:
- header comercial cu promo bar și categorii rapide
- homepage cu hero puternic, cards de categorii, servicii și membership
- pagină de produse mai curată, orientată pe shopping
- pagină de contact modernizată
- schimbare nativă de limbă: RO / EN / PL
- paletă portocalie păstrată

## Rulare
```bash
npm install
npm run dev
```

## Observație
Arhiva inițială conținea `node_modules` instalat pentru Windows. Pentru build local pe alt sistem de operare, rulează din nou `npm install` ca să fie refăcute dependențele native corecte.
